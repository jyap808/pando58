# Bill of Materials

This BOM is for the basic FR4 PCB setup with 2x PCBs (one for each half), 2x switch plates, and 2x bottom plates. Other setups like 3D printed cases are possible but not covered here.

## Components

### Required

#### Electronics

- 2x RP2040 Zero (Waveshare or [clones](https://www.amazon.com/dp/B0DXL1KDY7?tag=fdb-20), one for each half)
- 58x Diodes (1N4148W SMT or [1N4148 through-hole](https://www.amazon.com/dp/B079KJ91JZ?tag=fdb-20))
- 2x RJ45 ports ([Molex 438600001](https://www.digikey.com/en/products/detail/molex/0438600001/410212) or clones with same footprint)
- 2x Pando58 PCBs (left and right, not reversible)
- 4x 9-pin male header pins ([for RP2040 Zero MCU](https://www.amazon.com/dp/B07BRYLMK8?tag=fdb-20), 2 rows per half)

#### Switches & Keycaps

- 58x MX switches
- 58x keycaps

#### Hardware

- 10x M2 8mm brass standoffs ([5 per half](https://www.amazon.com/dp/B0FLGTHV32?tag=fdb-20))
- 20x M2 4mm screws ([10 per half](https://www.amazon.com/dp/B082XXVGT3?tag=fdb-20))
- 2x Pando58 switch plates
- 2x Pando58 bottom plates

#### Cables

- 1x USB-C cable (to connect to computer)
- 1x RJ45 cable ([for interconnect](https://www.amazon.com/dp/B0CMYYN58G?tag=fdb-20))

### Optional

#### Electronics

- 4x 9-pin female header sockets ([for removable MCU installation](https://www.amazon.com/dp/B07BRYLMK8?tag=fdb-20), 2 rows per half)

#### Switches & Keycaps

- 58x [MX hotswap sockets](https://www.amazon.com/dp/B0B4W9YMGM?tag=fdb-20)

#### Hardware

- 8x 3M SJ5302 bumpons ([4 per half](https://www.digikey.com/en/products/detail/3m/SJ5302/2345107))

### Assembly Tools

- Soldering iron and solder
- Screwdrivers
- Kapton tape (optional)
- Flush cutters (optional)
- Tweezers (optional)
- Multimeter (optional)

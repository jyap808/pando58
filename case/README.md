# Pando58 Case

![Pando58 Case](case-photo.jpg)

3D printable case for Pando58 keyboard.

This replaces the default FR4 bottom piece. The assembled keyboard uses: switch plate + PCB + bottom case (this piece).

**Note:** This model represents the **LEFT side** only. To print the right side, mirror the model in your slicer.

## Printing

The ready-to-print STL file is available on [Printables](https://www.printables.com/model/1647072-pando58-case).
